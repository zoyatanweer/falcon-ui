Input

Inputs are used to take the information typed by the user.

Textarea

The textarea is a multi-line text input control. It is widely used in a form, to collect user inputs like comments or feedbacks.
To use textarea, use the class textbox.

Label Input

Label Input has label present on top of the input field. This signifies the user of the value to be entered in respective fields.

Placeholder Input

Placeholder Input has a placeholder value inside the input field.
