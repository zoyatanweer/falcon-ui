Badge

We see badges everywhere. Whether it be messages, notifications etc.

Badge on avatar

Badge on avatar shows the current state of the user (online, offline or disabled) on any app . It has left and right positions.
There are four sizes of badges: xlg, lg, md and sm.
To use badges, use the class badge-(size)-(position) .Replace the size and postion with specified given below.

Badge on icon

Badge on icon shows the number of notifications on an icon.
