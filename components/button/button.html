Button

Buttons are also called as call to action. Here are different types of buttons. You need to add respective classes, and then you are good to use them.

Contained Button

Contained buttons are mainly to create a maximum impact on the user. Use the class btn btn-primary or btn-secondary to generate desired button. For creating hovering effect, use the box-shadow class.

Outlined Button

Outlined buttons can be created by using the class btn-outline- and then adding the required color of button.

Link Button

For attaching a link at the button, use the link button. Use the class btn-link.

Icon Button

Some buttons cannot be understood just by looking at their icons. So, for those buttons, use the icon button as shown below. Use the class btn-icons.

Floating Action Button

Floating action buttons can be customized by adding the font-awesome icon of your choice. Use the class btn-floating-action.
