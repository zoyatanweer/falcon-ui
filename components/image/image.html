Image

Images can be round or square shaped depending upon the choice. They can also be made responsive to fit the parent's width.

Round Image

Round shape images can be created by using the class round-img- and then using xl or lg as per required size.

round image xl size = 35rem * 35rem
round image lg size = 30rem * 30rem

Square Image

Square shape images can be created by using the class square-img- and then using xl or lg as per requirement.

square image xl size = 40rem * 40rem
square image lg size = 30rem * 30rem

Responsive Image

Responsive images can be created by using the class responsive-img- . Resize the window to see the effect.
