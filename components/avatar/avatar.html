Avatar

Avatar can be used to view a profile image of a person.
It has 5 Sizes: Avatar-xlg(15rem), Avatar-lg(13rem), Avatar-md(11rem), Avatar-sm(9rem), Avatar-xsm(7rem).

Round avatar

Round avatar can be created by using the class avatar-round and adding the classes according to the size.

Square avatar

Square avatar can be created by using the class avatar-square and adding the classes according to the size.
